# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-11-19

### Added
- Initial release
- `pull` command to sync articles from Intercom to local files
- `push` command to sync local files to Intercom
- `init` command to initialize help-docs directory structure
- Support for i18n with multiple locales
- Front matter metadata in markdown files
- Bidirectional sync between Intercom and local files
- Environment variable support for access token

### Features
- Pull all articles or specific article by ID
- Push all articles or specific file
- Automatic directory structure creation
- Translation management via front matter
- Article grouping by locale and collection
